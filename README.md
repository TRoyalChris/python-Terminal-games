# Python Terminal Games Collection 🎮

A collection of lightweight, interactive terminal-based games built using pure Python. This repository showcases structural loops, condition validation, and dynamic state arrays.

## Games Included

### 1. Blackjack Engine
* **Features:** Implements real-time player hand logic, rule validation for splitting/ace reduction adjustments (mitigating 11 down to 1), and automated dealer constraints (CPU stands on 17).
* **Core Concepts:** Procedural design, multi-threaded text loops, and list manipulation.

### 2. Hangman Engine
* **Features:** Tracks dynamic user guess historical data, evaluates character string collisions, and tracks remaining player life pools with matching ASCII art outputs.
* **Core Concepts:** Input validation, array filtering, and external resource asset linking.

### 3. Rock, Paper, Scissors Engine 
* **Features:** An interactive implementation of the classic hand game. It uses automated integer choice mapping, generates random CPU hands on the fly, and uses a structured conditional evaluation matrix to determine wins, losses, or ties.
* **Core Concepts:** Random distribution tracking, logic tree branching (`if/elif/else`), evaluation matrices, and modular code communication.
